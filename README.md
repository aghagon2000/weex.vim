# weex.vim
Weex syntax file for Vim.

Vim 的 [Weex][] 语法高亮插件

## 如何安装

使用 [Vundle][vundle]:

```
Plugin 'yuezk/weex.vim'
```

使用 [vim-plug][plug]:

```
Plug 'yuezk/weex.vim'
```

使用 [NeoBundle][bundle]:

```
NeoBundle 'yuezk/weex.vim'
```
